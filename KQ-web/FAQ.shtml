Frequently Asked Questions

1. What is KQ?
KQ is a console-style role-playing game for the PC. It will run on DOS, Linux and Windows (95/NT and above)
2. Why is it here?
KQ was abandoned by its original creator, for personal reasons. Because KQ was open source, we can host it here on Sourceforge
3. Is KQ finished?
According the author, the engine is complete, but some more scripting/level design needs to be done before KQ is finally done
4. What do I need to run KQ?
You will need the Allegro 4.x library, JGMOD (for music) and Lua (for scripting)
5. Can I contribute?
Please do. Just contact me
